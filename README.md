## English

## Project Overview

**Travel to Seychelles** is a responsive multi-page website created as a frontend practice project.
The website presents a fictional travel agency that offers information about the Seychelles, travel services, a 10-day travel itinerary, an image gallery and a contact page.

The goal of this project was to practice clean page structure, responsive design, navigation, image presentation and basic JavaScript functionality.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Design
* Media Queries
* Flexbox

## Features

* Responsive multi-page website
* Mobile-friendly navigation
* Image gallery with button navigation
* Swipe support for mobile devices
* Contact form layout
* 10-day travel itinerary page
* Separate pages for services, islands, gallery and contact
* Structured layout with reusable design elements

## Pages

* **Home** – Introduction to the fictional travel agency
* **Services** – Overview of travel-related services
* **Islands** – Information about different islands of the Seychelles
* **Travel Plan** – Example 10-day travel itinerary
* **Gallery** – Image gallery with JavaScript functionality
* **Contact** – Contact form layout and agency information

## What I Practiced

During this project, I practiced:

* Building a complete static website with multiple pages
* Creating a consistent layout across different pages
* Writing structured HTML
* Styling pages with CSS
* Creating responsive layouts for smaller screens
* Using JavaScript for an interactive image gallery
* Organizing project files into folders
* Improving the user experience with simple navigation and visual structure

## Projektübersicht

**Travel to Seychelles** ist eine responsive mehrseitige Webseite, die als Frontend-Übungsprojekt erstellt wurde.
Die Webseite präsentiert ein fiktives Reisebüro, das Informationen über die Seychellen, Reisedienstleistungen, einen 10-tägigen Reiseplan, eine Bildergalerie und eine Kontaktseite anbietet.

Ziel dieses Projekts war es, Seitenstruktur, responsives Design, Navigation, Bilddarstellung und einfache JavaScript-Funktionalität zu üben.

## Verwendete Technologien

* HTML5
* CSS3
* JavaScript
* Responsives Webdesign
* Media Queries
* Flexbox

## Funktionen

* Responsive mehrseitige Webseite
* Mobilfreundliche Navigation
* Bildergalerie mit Schaltflächen
* Swipe-Unterstützung für mobile Geräte
* Kontaktformular-Layout
* 10-tägiger Reiseplan
* Separate Seiten für Dienstleistungen, Inseln, Galerie und Kontakt
* Strukturierter Aufbau mit wiederverwendbaren Designelementen

## Seiten

* **Startseite** – Einführung in das fiktive Reisebüro
* **Dienstleistungen** – Übersicht der angebotenen Reisedienstleistungen
* **Inseln** – Informationen über verschiedene Inseln der Seychellen
* **Reiseplan** – Beispiel für einen 10-tägigen Reiseverlauf
* **Galerie** – Bildergalerie mit JavaScript-Funktionalität
* **Kontakt** – Kontaktformular und Informationen zum Reisebüro
